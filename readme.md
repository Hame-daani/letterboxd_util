# Letterboxd Util
it's a browser extension for both **chrome** and **firefox**.  
it will assist you with downloading movies directly from the movie page on letterboxd.  
> more sites will be support soon...


## Supported Sites
- 30Nama
- Digimoviez
- Lime Torrent
- Torrent Galaxy
- 1337x Torrent
